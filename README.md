# imdb-sentiment-fastapi-main
Sentimental analysis project

A FastAPI application that analyzes movie review sentiments using Hugging Face Transformers and TMDb's review API. It fetches reviews by movie title, processes them with a pre-trained model, and presents clear visual sentiment cues with information about the movie. Built with nlptown/bert-base-multilingual uncased-sentiment and deployed seamlessly on Render and Railway for fast, scalable access
